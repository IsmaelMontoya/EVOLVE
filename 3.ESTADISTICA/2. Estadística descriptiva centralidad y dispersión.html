Media Aritmética

La media es el promedio de todos los valores. Suma todos los datos y divide por la cantidad.

x̄ = (Σ xᵢ) / n = (x₁ + x₂ + ... + xₙ) / n 
salarios = [28, 32, 35, 30, 31, 
            29, 150, 33, 27, 34]

# Con numpy
media = np.mean(salarios)
print(f"Media: {media:.2f}k€")

# Con pandas
df['salario'].mean()
Resultado: 42.90 k€

Características:

✅ Usa toda la información disponible
✅ La más conocida y utilizada
❌ Muy sensible a outliers
❌ Puede no representar bien datos asimétricos
3

Mediana

La mediana es el valor central cuando los datos están ordenados. Divide la distribución en dos mitades iguales.

salarios = [28, 32, 35, 30, 31, 
            29, 150, 33, 27, 34]

# Ordenar primero
salarios_ord = sorted(salarios)
# [27,28,29,30,31,32,33,34,35,150]

# Con numpy/pandas
mediana = np.median(salarios)
print(f"Mediana: {mediana:.2f}k€")
Resultado: 31.50 k€

Características:

✅ Robusta ante outliers (no se ve afectada)
✅ Mejor representación en distribuciones asimétricas
❌ No usa toda la información (solo posiciones)
❌ Menos eficiente en distribuciones normales

Moda

La moda es el valor que más se repite en el conjunto de datos.

tallas = ['M', 'L', 'M', 'S', 'M', 
          'L', 'XL', 'M', 'S', 'M']

# Con pandas
moda = df['talla'].mode()[0]
print(f"Moda: {moda}")

# Frecuencias
df['talla'].value_counts()
M: 5 veces
L: 3 veces
S: 2 veces

Tipos de distribuciones:

  • Unimodal: Un solo pico (una moda)
  • Bimodal: Dos picos (dos modas)
  • Multimodal: Varios picos

Uso de la moda:

✅ Única medida para variables categóricas
✅ Útil para identificar el valor más frecuente
❌ Puede no existir (valores únicos)

Percentiles y Cuartiles

Los percentiles dividen los datos ordenados en 100 partes iguales. Los cuartiles son casos especiales.

Cuartil Percentil Significado
Q1 P25 25% de los datos están por debajo
Q2 P50 50% de los datos (= Mediana)
Q3 P75 75% de los datos están por debajo 
# Tiempos de respuesta de servidor (ms)
tiempos = [45, 52, 48, 67, 51, 120, 49, ...]

p25 = np.percentile(tiempos, 25)  # Q1
p50 = np.percentile(tiempos, 50)  # Mediana
p75 = np.percentile(tiempos, 75)  # Q3
p95 = np.percentile(tiempos, 95)  # P95 (SLA común)
Uso práctico en SLAs: P50 representa el caso típico, mientras que P90, P95 y P99 muestran el rendimiento en los peores casos.

Medidas de Dispersión

Las medidas de dispersión indican cuánto varían los datos respecto al centro.

¿Por qué son importantes?

Dos conjuntos de datos pueden tener la misma media, pero diferir enormemente en su variabilidad.

Empresa A: Salarios muy homogéneos

Todos ganan entre 28k€ y 35k€

Media: 31.4k€
Dispersión: Baja

Empresa B: Gran desigualdad

Salarios desde 35k€ hasta 150k€

Media: 75.0k€
Dispersión: Alta

Rango e IQR

Rango

Diferencia entre el máximo y el mínimo.

Rango = x_máx - x_mín 
datos = [15, 18, 22, 25, 28, 100]
rango = max(datos) - min(datos)
# Rango = 85
✅ Muy fácil de calcular
❌ Extremadamente sensible a outliers

Rango Intercuartílico (IQR)

Rango del 50% central de los datos.

IQR = Q3 - Q1 
Q1 = np.percentile(datos, 25)
Q3 = np.percentile(datos, 75)
IQR = Q3 - Q1

# Detectar outliers
lim_inf = Q1 - 1.5 * IQR
lim_sup = Q3 + 1.5 * IQR
✅ Robusto ante outliers
✅ Usado para detectar anomalías

Varianza

La varianza mide la dispersión promedio al cuadrado respecto a la media.

s² = Σ(xᵢ - x̄)² / (n-1)
¿Por qué n-1? Se llama corrección de Bessel. Al estimar la varianza poblacional desde una muestra, dividir por n subestima el valor real. Dividir por n-1 corrige este sesgo. 
datos = [28, 32, 35, 30, 31, 29, 33, 34, 27, 31]

# Varianza muestral (n-1)
varianza = np.var(datos, ddof=1)  # ddof=1 para muestral
print(f"Varianza: {varianza:.2f}")

# Pandas usa ddof=1 por defecto
df['salario'].var()

Nota sobre ddof (delta degrees of freedom):

  • ddof=0: Varianza poblacional (divide por N)
  • ddof=1: Varianza muestral (divide por N-1) - Pandas por defecto

Desviación Estándar

La desviación estándar es la raíz cuadrada de la varianza. Ventaja: está en las mismas unidades que los datos originales.

σ = √(σ²) 
datos = [28, 32, 35, 30, 31, 29, 33, 34, 27, 31]

media = np.mean(datos)  # 31.0 k€
std = np.std(datos, ddof=1)  # 2.45 k€

print(f"La mayoría de salarios están entre {media-std:.1f} y {media+std:.1f} k€")

Regla Empírica (68-95-99.7) para distribuciones normales:

  • 68% de los datos están entre μ - 1σ y μ + 1σ
  • 95% de los datos están entre μ - 2σ y μ + 2σ
  • 99.7% de los datos están entre μ - 3σ y μ + 3σ
11

Coeficiente de Variación (CV)

El CV expresa la desviación estándar como porcentaje de la media. Permite comparar variabilidad de datasets con diferentes escalas.

CV = (s / x̄) × 100% 
empresa_A = [30, 32, 28, 35, 31, 33, 29, 34, 30, 32]
empresa_B = [45, 80, 35, 120, 55, 90, 40, 150, 60, 75]

for nombre, datos in [("A", empresa_A), ("B", empresa_B)]:
    media = np.mean(datos)
    std = np.std(datos, ddof=1)
    cv = (std / media) * 100
    print(f"{nombre}: Media={media:.1f}k€, CV={cv:.1f}%")

Empresa A

Media: 31.4 k€
CV: 7.0%

Baja variabilidad - salarios homogéneos

Empresa B

Media: 75.0 k€
CV: 47.7%

Alta variabilidad - gran desigualdad

Pandas: El Método .describe()

El método .describe() calcula todas las estadísticas descriptivas de una vez.

import pandas as pd
import numpy as np

# Crear dataset de ejemplo
np.random.seed(42)
df = pd.DataFrame({
    'edad': np.random.randint(20, 60, 100),
    'salario': np.random.normal(45000, 15000, 100),
    'experiencia': np.random.randint(0, 30, 100)
})

# Resumen estadístico completo
df.describe()

Métodos Pandas - Referencia Rápida

Tendencia Central

df['col'].mean()    # Media
df['col'].median()  # Mediana
df['col'].mode()    # Moda

Dispersión

df['col'].std()     # Desv. Estándar
df['col'].var()     # Varianza
df['col'].min()     # Mínimo
df['col'].max()     # Máximo

Percentiles

df['col'].quantile(0.25)  # Q1
df['col'].quantile(0.50)  # Mediana
df['col'].quantile(0.75)  # Q3
df['col'].quantile(0.95)  # P95

Análisis por Grupos

df.groupby('categoria')['valor'].agg([
    'count', 'mean', 'median', 'std'
])
Tip: Pandas usa ddof=1 por defecto para .std() y .var() (varianza muestral).

Guía de Selección de Medidas

¿Cuándo usar cada medida de tendencia central?

Situación Usar Razón
Datos simétricos sin outliers Media Usa toda la información
Datos con outliers Mediana Robusta a extremos
Distribución sesgada Mediana Representa mejor el "típico"
Variables categóricas Moda Única opción
Tiempos de respuesta Mediana + P95/P99 Típico + peor caso
Precios de viviendas Mediana Siempre hay outliers de lujo

Conclusiones Clave

Lo esencial de esta clase:

  • No existe una medida perfecta. La elección depende del contexto y tipo de datos.
  • Media y mediana diferentes = alerta. Indica outliers o asimetría.
  • La dispersión es tan importante como el centro. Dos datasets con la misma media pueden ser muy diferentes.
  • CV permite comparar variabilidad entre datasets con diferentes escalas.
  • Pandas hace todo fácil: .describe(), .mean(), .median(), .std()
