Asimetría (Skewness)

La asimetría mide la falta de simetría de una distribución respecto a su media.

Simétrica

⛰️

Skewness ≈ 0

Media = Mediana

Sesgada Derecha

📈

Skewness > 0

Mediana < Media
Cola larga a la derecha

Sesgada Izquierda

📉

Skewness < 0

Media < Mediana
Cola larga a la izquierda

from scipy import stats

skewness = stats.skew(datos)
print(f"Skewness: {skewness:.2f}")
3

Interpretación de Skewness

Valor de Skewness Interpretación
≈ 0 Distribución simétrica
0 a 0.5 Ligeramente sesgada a la derecha
0.5 a 1 Moderadamente sesgada a la derecha
> 1 Muy sesgada a la derecha
-0.5 a 0 Ligeramente sesgada a la izquierda
-1 a -0.5 Moderadamente sesgada a la izquierda
< -1 Muy sesgada a la izquierda
💡 Ejemplos prácticos:
• Ingresos/salarios: Sesgada derecha (pocos muy ricos)
• Edad de jubilación: Sesgada izquierda (mayoría se jubila tarde)
• Altura humana: Aproximadamente simétrica
4

Curtosis (Kurtosis)

La curtosis mide el peso de las colas de la distribución y la altura del pico central.

Platicúrtica

⏜

Curtosis < 0

  • Pico aplanado
  • Colas ligeras
  • Menos outliers

Mesocúrtica

⌒

Curtosis ≈ 0

  • Distribución Normal
  • Referencia estándar
  • Curtosis = 0

Leptocúrtica

⋀

Curtosis > 0

  • Pico alto
  • Colas pesadas
  • Más outliers
curtosis = stats.kurtosis(datos)  # Exceso de curtosis
print(f"Curtosis: {curtosis:.2f}")
Se resta 3 para que la distribución normal tenga curtosis = 0. Esto se llama "exceso de curtosis".
5

Impacto en la Elección de Medidas

La forma de la distribución determina qué medidas descriptivas son más apropiadas.

Forma de Distribución Medida Central Medida de Dispersión
Simétrica, sin outliers Media Desviación estándar
Sesgada (cualquier dirección) Mediana IQR
Alta curtosis (muchos outliers) Mediana IQR
Baja curtosis Media o Mediana IQR o Std (ambos válidos)
Regla práctica: Si skewness > 0.5 o curtosis > 1, considera usar mediana en lugar de media.

¿Qué es un Outlier?

Un outlier (valor atípico) es una observación que se desvía significativamente del patrón general de los datos.

⚠️ Outliers como ERRORES

Deben corregirse o eliminarse

  • Edad = -5 años (imposible)
  • Precio = 0€ (error de entrada)
  • Temperatura = 500°C (sensor roto)

✅ Outliers LEGÍTIMOS

Valores raros pero válidos

  • Salario de CEO en empresa
  • Ventas del Black Friday
  • Tiempos de respuesta en caída

¡Los outliers pueden ser la información más valiosa!

Detección de fraude, comportamiento anómalo, oportunidades de negocio...

Método IQR (Rango Intercuartílico)

El método más robusto y ampliamente usado. Define outliers como valores fuera de los "bigotes" del boxplot.

Límite Inferior = Q1 - 1.5 × IQR
Límite Superior = Q3 + 1.5 × IQR 
# Calcular límites
Q1 = np.percentile(datos, 25)
Q3 = np.percentile(datos, 75)
IQR = Q3 - Q1

lim_inf = Q1 - 1.5 * IQR
lim_sup = Q3 + 1.5 * IQR

# Detectar outliers
outliers = (datos < lim_inf) | (datos > lim_sup)
print(f"Outliers: {outliers.sum()}")

Características del método IQR:

✅ Robusto ante distribuciones sesgadas
✅ No asume normalidad de los datos
✅ Visualización clara con boxplot
❌ Puede ser conservador (detecta muchos)
Variante IQR 3×: Usar factor 3.0 en lugar de 1.5 para detectar solo outliers extremos.
8

Método Z-score

El Z-score mide cuántas desviaciones estándar está un valor de la media.

Z = (x - μ) / σ

Regla común: valores con |Z| > 3 se consideran outliers

# Calcular Z-scores
media = np.mean(datos)
std = np.std(datos, ddof=1)

z_scores = (datos - media) / std

# Detectar outliers
outliers_z = np.abs(z_scores) > 3

print(f"Outliers Z-score: {outliers_z.sum()}")

Características del Z-score:

✅ Intuitivo: basado en σ
✅ Fácil de interpretar
❌ Asume distribución normal
❌ Sensible a outliers (usa media y std)
Paradoja: Los outliers afectan la media y std, lo que puede dificultar su propia detección.
9

¿Cuándo Usar Cada Método?

Método Ventajas Cuándo Usar
IQR (1.5×) Robusto a distribuciones sesgadas Datos con distribución no normal o asimétrica
Z-score (3σ) Intuitivo, basado en desviación estándar Datos aproximadamente normales y simétricos
IQR (3×) Detecta solo extremos Cuando solo interesan outliers muy extremos
Z-score (2σ) Más sensible Exploración inicial o control de calidad estricto

Recomendación General

Usa IQR como método predeterminado, especialmente si no estás seguro de la forma de la distribución. El Z-score funciona bien solo cuando los datos son aproximadamente normales.

Estrategias de Tratamiento de Outliers

Una vez detectados, hay varias opciones para tratarlos:

1. ELIMINAR

  • Quitar del dataset
  • Cuándo: Error claro de datos
  • ⚠️ Precaución: Pérdida de información

2. TRANSFORMAR

  • Aplicar log, sqrt, Box-Cox
  • Cuándo: Reducir asimetría
  • ⚠️ Precaución: Cambia interpretación

3. IMPUTAR

  • Reemplazar por mediana
  • Cuándo: Mantener tamaño de muestra
  • ⚠️ Precaución: Introduce sesgo

4. WINSORIZAR

  • Cap en límites IQR
  • Cuándo: Reducir impacto sin eliminar
  • ⚠️ Precaución: Modifica distribución

5. MANTENER

  • Dejar como está
  • Cuándo: Información válida importante
  • ✅ Acción: Usar medidas robustas

6. SEPARAR

  • Analizar por separado
  • Cuándo: Outliers son su propio segmento
  • ✅ Acción: Crear variable indicadora
11

Árbol de Decisión para Tratamiento

Sigue este proceso para decidir cómo tratar cada outlier:

¿Es un ERROR de datos? │ ├── SÍ → ¿Puedo corregirlo? │ ├── SÍ → CORREGIR │ └── NO → ELIMINAR │ └── NO → ¿El outlier es importante para el análisis? │ ├── SÍ → MANTENER │ (separar análisis o crear indicador) │ └── NO → ¿Necesito los datos para modelar? ├── SÍ → TRANSFORMAR o IMPUTAR └── NO → WINSORIZAR o ELIMINAR
Principio fundamental: Siempre documenta tu decisión y justifica por qué elegiste ese tratamiento. La transparencia es clave en Data Science.

Implementación de Estrategias

import numpy as np
import pandas as pd

# Detectar outliers con IQR
Q1, Q3 = df['valor'].quantile([0.25, 0.75])
IQR = Q3 - Q1
lim_inf = Q1 - 1.5 * IQR
lim_sup = Q3 + 1.5 * IQR
mask_outliers = (df['valor'] < lim_inf) | (df['valor'] > lim_sup)

# 1. ELIMINAR
df_sin_outliers = df[~mask_outliers].copy()

# 2. TRANSFORMAR (Log)
df['valor_log'] = np.log(df['valor'] + 1)  # +1 para valores positivos

# 3. IMPUTAR con mediana
df['valor_imputado'] = df['valor'].copy()
df.loc[mask_outliers, 'valor_imputado'] = df['valor'].median()

# 4. WINSORIZAR (cap en límites)
df['valor_winsorizado'] = df['valor'].clip(lower=lim_inf, upper=lim_sup)

# 5. MANTENER + crear indicador
df['es_outlier'] = mask_outliers.astype(int)

Concepto Definición Valores de Referencia
