Variables Aleatorias

Una variable aleatoria es una función que asigna un valor numérico a cada resultado de un experimento aleatorio.

Variables Discretas

Valores contables (finitos o saltos).

  • Número de clicks en un anuncio
  • Productos vendidos hoy
  • Usuarios activos
Usan PMF (Probability Mass Function)

Variables Continuas

Cualquier valor en un intervalo.

  • Tiempo de carga de una web
  • Temperatura del servidor
  • Ingresos por venta
Usan PDF (Probability Density Function)

Funciones de Probabilidad

Las fórmulas matemáticas que describen cómo se comportan las probabilidades.

Función de Densidad (PDF)

Para variables continuas. La probabilidad es el área bajo la curva.

$$P(a \leq X \leq b) = \int_a^b f(x)\,dx$$

Función Acumulada (CDF)

Probabilidad de que la variable sea menor o igual a un valor x.

$$F(x) = P(X \leq x) = \int_{-\infty}^{x} f(t)\,dt$$
En Python (scipy.stats):
dist.pdf(x) / dist.pmf(k) para densidad/masa.
dist.cdf(x) para probabilidad acumulada (muy útil para p-values).

Distribución Uniforme

Todos los valores en un intervalo tienen la misma probabilidad de ocurrir.

$$f(x) = \frac{1}{b - a}, \quad a \leq x \leq b$$

Propiedades

  • Media: (a + b) / 2
  • Varianza: (b - a)² / 12

Casos de Uso

  • Generación de números aleatorios
  • Simulaciones Monte Carlo
  • Asignación en A/B Testing
from scipy import stats
uniforme = stats.uniform(loc=0, scale=10) # U(0, 10)
prob = uniforme.cdf(3) # P(X < 3) = 0.3

Distribución Normal (Gaussiana)

La reina de las distribuciones. Forma de campana simétrica.

$$f(x) = \frac{1}{\sigma\sqrt{2\pi}}\, e^{-\dfrac{(x-\mu)^2}{2\sigma^2}}$$

Parámetros

  • μ (mu): Media (centro)
  • σ (sigma): Desviación estándar (ancho)

Por qué es importante

Muchos fenómenos naturales y errores de medición siguen esta forma (Teorema del Límite Central).

6

Regla Empírica 68-95-99.7

Porcentajes de datos dentro de desviaciones estándar de la media.

68%

Entre $\mu \pm 1\sigma$

95%

Entre $\mu \pm 2\sigma$

99.7%

Entre $\mu \pm 3\sigma$

7

Z-Score (Estandarización)

Convierte cualquier normal a la Normal Estándar (μ=0, σ=1).

$$Z = \frac{X - \mu}{\sigma}$$

Nos dice cuántas desviaciones estándar está un valor de la media.

  • Z = 0 → Valor es igual a la media
  • Z = 1 → Valor está a 1σ por encima
  • Z = -2 → Valor está a 2σ por debajo
# Calcular probabilidad P(X > 120)
# para N(100, 15)
mu, sigma = 100, 15
prob = 1 - stats.norm.cdf(120, mu, sigma)
# Resultado: 0.091 (9.1%)

Distribución Binomial

Modela el número de éxitos en n ensayos independientes.

$$P(X = k) = \binom{n}{k} p^k (1-p)^{n-k}$$

Parámetros

  • n: Número de ensayos
  • p: Probabilidad de éxito

Media: μ = n·p

Casos de Uso

  • Conversiones de 100 visitas
  • Emails abiertos de 1000 enviados
  • Productos defectuosos en un lote
# Probabilidad de 15 ventas en 500 visitas (conv=2.5%)
stats.binom(n=500, p=0.025).pmf(15)
9

Distribución de Poisson

Modela el número de eventos en un intervalo de tiempo o espacio.

$$P(X = k) = \frac{\lambda^k e^{-\lambda}}{k!}$$

Parámetros

  • λ (lambda): Tasa promedio de eventos

Media: μ = λ
Varianza: σ² = λ

Casos de Uso

  • Llamadas a call center por hora
  • Visitas web por minuto
  • Errores de código por 1000 líneas
10

Distribución Exponencial

Modela el tiempo entre eventos de un proceso Poisson.

$$f(x) = \lambda e^{-\lambda x}, \quad x \geq 0$$

Propiedades

  • Media: 1 / λ
  • Sin memoria: Lo que pasó antes no afecta al futuro.

Casos de Uso

  • Tiempo hasta el próximo fallo
  • Tiempo entre llegadas de clientes
  • Duración de una llamada (a veces)
11

Distribución Log-Normal

Para variables cuyo logaritmo es normal. Típica en datos positivos con sesgo a la derecha.

Características

  • Solo valores positivos
  • Cola larga a la derecha (asimetría positiva)
  • Media > Mediana

Ejemplos Reales

  • Ingresos y Salarios
  • Precios de activos financieros
  • Tiempo de permanencia en sitio web

Resumen Rápido

Distribución Tipo Pregunta Clave
Uniforme Continua ¿Cualquier valor es igual de probable?
Normal Continua ¿Es simétrica y resultado de muchos factores?
Binomial Discreta ¿Cuántos éxitos en n intentos?
Poisson Discreta ¿Cuántos eventos en un intervalo?
Exponencial Continua ¿Cuánto tiempo hasta el próximo evento?
Log-Normal Continua ¿Variable positiva con sesgo (ingresos)?

Ejercicios Prácticos

1. E-commerce

500 visitas, conv=2.5%. ¿Probabilidad de ≥15 ventas?

Dist: Binomial(n=500, p=0.025)

2. Call Center

20 llamadas/hora. ¿Probabilidad de >25 llamadas?

Dist: Poisson(λ=20)

3. Tiempos Respuesta

Normal(μ=200ms, σ=30ms). ¿% > 250ms?

Dist: Normal

4. Fiabilidad

Fallo cada 30 días. ¿P(durar > 60 días)?

Dist: Exponencial(λ=1/30)

Conclusiones

Puntos Clave:

  • Las distribuciones son modelos de la realidad, no la realidad misma.
  • Identificar la distribución correcta es el 80% del trabajo.
  • Python (scipy.stats) facilita todos los cálculos complejos.
  • La Normal es fundamental, pero en negocios (ingresos, conversiones) a menudo usamos otras.
