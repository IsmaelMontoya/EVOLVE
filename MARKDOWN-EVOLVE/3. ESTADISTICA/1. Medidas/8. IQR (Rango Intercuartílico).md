**Fórmula:** $IQR = Q3 - Q1$

|Símbolo|Significado|
|---|---|
|$Q3$|Tercer Cuartil = Percentil 75|
|$Q1$|Primer Cuartil = Percentil 25|

- **Qué hace:** Mide la distancia entre el 25% más barato y el 25% más caro. Aísla el 50% central.
- **Para qué se usa:** Ver dispersión real ignorando extremos. Base para diagramas de caja (boxplots).
- **Interpretación:** "El 50% central del mercado se mueve en una franja de IQR euros."

> [!example] Ejercicio Real Q3 = 300.000 €, Q1 = 200.000 € IQR = 300.000 - 200.000 = **100.000 €**
