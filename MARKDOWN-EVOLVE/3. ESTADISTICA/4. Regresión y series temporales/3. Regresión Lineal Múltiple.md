**Fórmula:** $\hat{Y} = \beta_0 + \beta_1 X_1 + \beta_2 X_2 + \cdots + \beta_p X_p + \epsilon$

- **Qué hace:** Evalúa el peso de varios factores simultáneamente.
- **Interpretación:** "A igualdad de metros cuadrados (_Ceteris Paribus_), un baño extra sube el precio 10.000€."

> [!example] Ejercicio Real Modelo: Precio = 20.000 + (1.200 · m²) + (5.000 · Hab) Casa de 80 m² con 3 habitaciones: 20.000 + (1.200 · 80) + (5.000 · 3) = 20.000 + 96.000 + 15.000 = **131.000 €**
