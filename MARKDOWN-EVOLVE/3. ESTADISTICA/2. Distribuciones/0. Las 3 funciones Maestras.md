#### PDF / PMF (Función de Densidad o Masa)

**Fórmula general:** $f(x) = P(X = x)$

- **PMF (Masa):** Para cosas contables (1, 2, 3 casas).
- **PDF (Densidad):** Para cosas continuas (precios, m²). Mide la "concentración" de probabilidad en un punto.
- **Para qué se usa:** Saber cuál es el valor más frecuente o "típico".

---

#### CDF (Función de Distribución Acumulada)

**Fórmula:** $F(x) = P(X \leq x) = \int_{-\infty}^{x} f(t)dt$

- **Qué hace:** Suma todas las probabilidades desde el inicio hasta el punto elegido.
- **Para qué se usa:** Calcular rangos y probabilidades de "hasta aquí".
- **Interpretación:** "El 80% de las casas cuestan 300.000€ o menos."

---

#### PPF (Percent Point Function / Inversa de la CDF)

**Fórmula:** $x = F^{-1}(p)$

- **Qué hace:** Es la inversa de la CDF. Le das un porcentaje y te devuelve el valor que marca ese límite.
- **Para qué se usa:** Establecer precios de corte o metas.
- **Interpretación:** "¿Qué precio máximo debo poner para ser más barato que el 90% del mercado?"
